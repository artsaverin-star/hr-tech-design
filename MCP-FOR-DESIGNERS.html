Claude + Figma

Пишешь задачу словами — Claude делает её прямо в твоей Figma: собирает экраны по картинке, ставит компоненты HRDS, тексты, проверяет тёмную тему.

Установка — один раз

Открой Терминал (Cmd+Пробел → набери «Терминал» → Enter) и вставляй блоки по очереди, каждый раз Enter.

1
Поставь Homebrew и Node.js
Попросит пароль Mac — вводи (при наборе он не виден, это нормально).
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile eval "$(/opt/homebrew/bin/brew shellenv)" brew install node
2
Поставь Claude
npm install -g @anthropic-ai/claude-code
3
Войди в свой аккаунт Claude
Откроется браузер — авторизуйся. Потом закрой Claude: Ctrl+C.
claude
4
Скачай проект
Создаст папку ~/hr-tech-design (= твоя домашняя папка).
cd ~ && git clone https://github.com/artsaverin-star/hr-tech-design.git
5
Установи
Дождись надписи «✓ Готово».
cd ~/hr-tech-design && ./setup.sh
6
Поставь плагин в Figma это не команда
Десктопная Figma → Plugins → Development → Import plugin from manifest… → откроется окно выбора файла → нажми Cmd+Shift+G, вставь путь, Enter → выбери manifest.json:
~/hr-tech-design/figma-desktop-bridge/manifest.json

Каждый раз

1
Включи плагин в Figma
Plugins → Development → HR TECH DESIGN → нажми Activate Bridge (лампочка зелёная). Не закрывай плагин.
2
Запусти Claude
cd ~/hr-tech-design && claude
3
Пиши задачу словами
Например: «Собери макет по этой картинке». Чтобы показать место — в Figma ПКМ по слою → Copy link → вставь ссылку в чат.

Если связь отвалилась

В чате Claude набери /mcp → выбери figma-hrtech → Reconnect
Не помогло — закрой Claude (Ctrl+C) и запусти заново: cd ~/hr-tech-design && claude
Путь у всех одинаковый: ~/hr-tech-design (~ = твоя домашняя папка).
